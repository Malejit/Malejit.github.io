ARQUITECTURA BASICA DE UN PROCESADOR

Es el diseño estructural que define cómo se organizan e interconectan los componentes principales del sistema para ejecutar instrucciones. Es como el plano que determina cómo funciona un computador internamente.

CARACTERISTICAS DE LA ARQUITECTURA BASICA DE UN PROCESADOR

  • Funcionamiento secuencial: La CPU busca, interpreta y ejecuta instrucciones una por una.
  • Estructura modular: Se compone de procesador, memoria, dispositivos de entrada/salida y almacenamiento.
  • Memoria compartida: Tanto datos como instrucciones se almacenan en la misma memoria principal (modelo Von Neumann).
  • Interconexión mediante buses: Todos los componentes se comunican a través de buses de datos, direcciones y control.
  • Automatización: Una vez iniciado un programa, el computador lo ejecuta sin intervención humana.
  • Flexibilidad: Puede ejecutar distintos programas sin necesidad de modificar el hardware.
